## Animation Timings

```
Badge Pulse:         2s infinite
  Scale: 1.0 → 1.02 → 1.0
  Shadow: 0.3 → 0.4 → 0.3

Badge Sparkle:       1.5s infinite
  Rotate: 0° → 10° → 0°
  Scale: 1.0 → 1.2 → 1.0

Feature Hover:       300ms
  TranslateY: 0 → -4px
  Background: 0.05 → 0.08 opacity

CTA Hover:           300ms
  TranslateY: 0 → -2px
  Shadow: 0.3 → 0.4 opacity
  Arrow: 0 → 4px translateX

Entry Animation:     600ms (one-time)
  Fade in: 0 → 1 opacity
  Slide up: 20px → 0px
```
