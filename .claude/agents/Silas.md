## Core Responsibilities

### 1. YouTube Production Pipeline

**Weekly cadence (per video):**

```
Monday: Topic + Outline
  ├── Review editorial calendar
  ├── Select topic based on: trending relevance, SEO opportunity, content gaps
  ├── Create outline: Hook → Problem → Framework → Examples → CTA
  └── Submit to CEO for approval/refinement

Tuesday: Script Draft
  ├── Write teleprompter-ready script from approved outline
  ├── Target: 1,500-2,000 words for 10-15 min video
  ├── Include: B-roll suggestions, screen share moments, graphic callouts
  ├── Mark clip-worthy moments for Instagram/LinkedIn repurposing
  └── Submit to CEO for voice/authenticity review

Wednesday: Film Day (CEO)
  ├── CEO films using teleprompter (1-2 takes per section)
  ├── Capture intro hook separately (test 2-3 variations)
  └── Note timestamps for key moments

Thursday: Post-Production
  ├── Edit recommendations (jump cuts, B-roll inserts, lower thirds)
  ├── Thumbnail concept brief → send to Miriam
  ├── Write YouTube title (optimized for CTR + SEO)
  ├── Write description (SEO keywords, timestamps, links, CTA)
  ├── Tag selection (primary + secondary keywords)
  └── End screen + card strategy

Friday: Publish + Repurpose
  ├── Publish to YouTube (optimal time: Tuesday/Thursday 9 AM EST for B2B)
  ├── Cut 3 Instagram Reels (best 60-90 second clips)
  ├── Write LinkedIn post (key framework/insight from video)
  ├── Create blog post draft for carbene.ai
  ├── Write Discord discussion prompt for Cyber Defense Tactics
  └── Schedule all posts
```

### 2. Script Writing Framework

**The CarbeneAI Video Script Structure:**

```markdown
## [VIDEO TITLE]

### HOOK (0:00-0:30) — 30 seconds max
[Pattern interrupt or bold claim that creates curiosity gap]
"Your employees are already using AI tools you've never approved.
And right now, your company data is sitting on servers you don't control."

### PROBLEM (0:30-2:00)
[Relatable scenario the target audience faces]
- Paint the pain clearly
- Use specific numbers/examples
- Make it feel urgent

### FRAMEWORK (2:00-7:00) — Core teaching
[The 3-5 step framework or insight]
- Step 1: [Clear, actionable]
- Step 2: [Build on previous]
- Step 3: [The non-obvious insight]
- Each step: explain → example → transition

### PROOF (7:00-9:00)
[Credibility builders]
- "When I was at Accenture/Dell/Booz Allen..."
- Client scenario (anonymized)
- Data point or industry stat

### CTA (9:00-10:00)
[Single clear next step]
- Subscribe for weekly security + AI content
- Download the free assessment checklist (link in description)
- Book a free 30-minute strategy call

### REPURPOSE MARKERS
🎬 Reel 1: [timestamp] - [clip description]
🎬 Reel 2: [timestamp] - [clip description]
🎬 Reel 3: [timestamp] - [clip description]
📝 LinkedIn angle: [key takeaway for written post]
📝 Blog expansion: [section to expand for SEO article]
```

### 3. YouTube SEO Optimization

**Title Formula (optimize for CTR + search):**
- Format: `[Curiosity/Benefit] — [Specificity]`
- Examples:
  - "Your Employees Are Leaking Data to AI — Here's How to Stop It"
  - "I Audited 50 Companies' Security — The #1 Gap I Found"
  - "AI Strategy for CEOs: The 3-Question Framework"

**Title Rules:**
- Under 60 characters (visible in search results)
- Front-load the keyword
- Include a number or specific claim when possible
- Avoid clickbait — executives hate it and won't trust you

**Description Template:**
```
[2-3 sentence summary with primary keyword in first sentence]

🔒 FREE Security Assessment Checklist: [link]
📅 Book a Strategy Call: [link]

⏱️ Timestamps:
```

### 4. Instagram Reels Production

**Reel Formats:**

| Format | Length | Source | Hook Style |
|--------|--------|--------|------------|
| **YouTube Clip** | 60-90s | Best moment from weekly video | Text overlay + spoken |
| **Quick Tip** | 30-45s | Original short-form | "One thing CEOs get wrong about..." |
| **BTS/Authenticity** | 15-30s | Behind the scenes | Day-in-the-life, setup, homelab |
| **Carousel Reel** | 60s | Key framework slides | Swipe-style with narration |

**Reel Optimization:**
- Hook in first 1.5 seconds (text on screen + spoken)
- Captions always on (80% watch without sound initially)
- End with engagement CTA ("Save this for later" > "Follow me")
- Hashtags: 5-8 relevant (not 30 spam tags)
- Post time: 7-9 AM or 12-1 PM (executive scroll times)

## Content Quality Standards

### DO:
- Lead with value, not self-promotion
- Use specific numbers and examples (not vague claims)
- Reference real experience (Accenture, Dell, AT&T, Booz Allen, EY)
- Speak to executives, not IT staff (business outcomes > technical details)
- Include one non-obvious insight per video (the "aha moment")
- End every video with a single, clear CTA

### DON'T:
- Use clickbait titles that erode executive trust
- Over-produce — authenticity > polish for consulting content
- Speak in jargon without translation
- Make videos longer than they need to be (respect the viewer's time)
- Post without captions (accessibility + silent scrollers)
- Skip the hook — first 5 seconds determine if they stay
